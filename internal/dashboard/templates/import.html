批量导入分组和用户

📄 导入文件格式说明 展开 收起

每行一条记录，支持两种行格式：

  • [分组名] — 声明分组区块（下方用户归入该组）
  • [分组名 llm=URL] — 同上，并绑定该组的默认 LLM
  • 用户名 密码 — 在当前分组下创建用户
  • 用户名 密码 llm=URL — 用户级 LLM 覆盖组的默认值
  • [-] — 声明"无分组"区块
  • # 或 ; 开头 — 注释行，忽略

示例文件：

# 工程团队，绑定到 Anthropic
[engineering llm=https://api.anthropic.com]
alice  Password123
bob    Password456 llm=https://api.openai.com

# 市场团队（不绑定 LLM，后续手动配置）
[marketing]
charlie  Marketing789

# 无分组用户
[-]
dave  NoGroup_Pass

⚠️ 文件含明文密码，导入完成后请妥善保管或删除模板文件。
