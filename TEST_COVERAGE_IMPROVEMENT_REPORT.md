### 1.2 新增测试文件

#### llm_handler_test.go (6个测试函数)
- `TestHandleLLMPage`: LLM 管理页面渲染
  - 基础页面加载
  - Flash 消息显示
  - 错误消息显示

- `TestHandleLLMCreateBinding`: LLM 绑定创建
  - 用户级绑定
  - 分组级绑定
  - 缺少 target_url 错误处理
  - 无效 bind_type 错误处理

- `TestHandleLLMDeleteBinding`: LLM 绑定删除
  - 删除已存在的绑定
  - 删除不存在的绑定

- `TestHandleLLMDistribute`: LLM 流量分配
  - 成功分配
  - 缺少参数错误处理

- `TestHandleDrainEnter`: 进入排水模式
  - 成功进入
  - 排水函数未配置
  - 排水失败错误处理

- `TestHandleDrainExit`: 退出排水模式
  - 成功退出
  - 排水函数未配置
  - 退出失败错误处理

#### handler_additional_test.go (6个测试函数)
- `TestHandleToggleActive`: 用户激活/停用
  - 切换用户状态
  - 验证状态变更

- `TestHandleResetPassword`: 密码重置
  - 成功重置密码
  - 缺少密码参数错误处理

- `TestHandleSetUserGroup`: 用户分组设置
  - 设置用户分组
  - 清除用户分组

- `TestHandleRevokeUserTokens`: 撤销用户令牌
  - 成功撤销令牌
  - TokenRepo 未配置错误处理

- `TestHandleSetQuota`: 分组配额设置
  - 设置配额限制

- `TestHandleDeleteGroup`: 分组删除
  - 成功删除分组
